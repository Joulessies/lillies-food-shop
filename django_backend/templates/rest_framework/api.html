Welcome to the Lillies Food Shop API

This API provides access to the Lillies Food Shop application data.

Main Endpoints

  • /api/menu/categories/ - Get all menu categories
  • /api/menu/items/ - Get all menu items (optionally filtered by category)
  • /api/health/ - Health check endpoint
  • /api/dashboard/stats/ - Dashboard statistics (authenticated users only)
  • /api/auth/login/ - User login
  • /api/auth/register/ - User registration

Authentication

Most endpoints require authentication. To authenticate, use the login endpoint to obtain an access token.
