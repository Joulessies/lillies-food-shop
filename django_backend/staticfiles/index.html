Lillies Food Shop API

Welcome to the backend service for the Lillies Food Shop application.

Available Endpoints

GET /api/ - API Root

The root of the API. Provides links to available endpoints.

GET /api/health/ - Health Check

Health check endpoint to verify the API is functioning properly.

GET /admin/ - Admin Interface

Django admin interface for backend management.

GET /api/menu/categories/ - Menu Categories

Get all available menu categories.

GET /api/menu/items/ - Menu Items

Get all menu items, optionally filtered by category.

Query Parameters:

  • category - Filter items by category ID

POST /api/auth/login/ - User Login

Authenticate user and get access token.

POST /api/auth/register/ - User Registration

Register a new user account.
